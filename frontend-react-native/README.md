# React Native UI Migration (Core)

This folder contains a React Native version of the core UI flow converted from the Flutter UI:

- Auth screen
- Home screen
- Cart screen
- Orders screen
- App-level theme toggle
- Bottom tab navigation

## Run

1. Open a terminal in `frontend-react-native`
2. Install packages:

```bash
npm install
```

3. Start Metro:

```bash
npm run start
```

4. Run app:

```bash
npm run android
```

or

```bash
npm run ios
```

## Notes

- This is a UI-first conversion.
- Backend API integration and remaining admin/driver/store-owner screens can be migrated next.
